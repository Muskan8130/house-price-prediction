# 🏡 House Price Prediction System

## Overview

This project predicts house prices in **Delhi, Noida, and Greater Noida** using Machine Learning models.

It provides:

- Accurate price prediction 💰
- Price range estimation 📊
- Confidence score 🎯
- Explanation of prediction 🧠

---

## ML Models Used

- Linear Regression
- Random Forest
- XGBoost (Best Model )

---

## Features

✔ Smart City → Locality dropdown
✔ Price in ₹ Lakh / Cr format
✔ Prediction range (±10%)
✔ Confidence score
✔ Explainable AI output
✔ Beautiful UI with Flask

---

## 🖥️ Tech Stack

- Python
- Flask
- Scikit-learn
- XGBoost
- Pandas
- HTML/CSS/JS
